# Markdown 代码

>如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：

`print()`函数


### 代码区块

>代码区块使用**4个空格**或者**一个制表符Tab**，实例如下：

    class Cat：
        def __init__(self, name):
            self.name = name


>也可以用```包裹一段代码，并指定一种语言（也可以不指定）：

```python
class Cat:
    def __init__(self, name):
        self.name = name
```
